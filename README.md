## Diário dos Sonos

O Diário dos Sonos é um Aplicativo Mobile que permite você registrar e controlar seus horários de sono (hora inicial e hora final).

### Tecnologias usadas
- React Native
- Expo

### Criação inicial do projeto

Os comandos usados para criar o projeto:

```
# Nome do projeto é 'diario_dos_sonos'

npx create-expo-app@latest
```
